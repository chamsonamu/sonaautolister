SonaAutoLister

SonaAutoLister is a personal application designed to help manage my Etsy shop more efficiently. This tool allows me to automatically upload products, update descriptions and images, adjust prices, and track inventory from one place.

Features

  • Bulk product uploads
  • Automatic inventory updates
  • Easy description and image management
  • Time-saving automation for Etsy sellers
